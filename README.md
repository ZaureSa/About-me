# **Во весь голос**

### Слушайте,

товарищи потомки,

агитатора,

горлана-главаря.

Заглуша

поэзии потоки,

я шагну

через лирические томики,

как живой

с живыми говоря.

Я к вам приду

в коммунистическое далеко

не так,

как песенно-есененный провитязь.

Мой стих дойдет

через хребты веков

и через головы

поэтов и правительств.
